# 政策解读包：青年人才住房补贴

## 一页摘要

这份政策面向在示例市首次就业、与本市企业签订三年以上劳动合同的全日制本科及以上毕业生，提供最长三年的住房补贴。硕士毕业生标准为每人每年 20000 元，本科毕业生为 10000 元，博士毕业生为 30000 元。[C2]

## 资格初判

当前用户画像下，结论应保守写为：**信息不足，部分条件可能符合**。

- 可能符合：应届硕士、拟入职本市企业、未享受过同类住房补贴。[C1][C3]
- 信息不足：社保需要连续缴纳满三个月，当前只提供了 1 个月记录；还需要确认本人及配偶在本市无自有住房。[C3]
- 不得承诺：是否通过审批取决于官方窗口复核和材料真实性。

## 行动清单

1. 补齐连续三个月社保缴纳记录。[C3]
2. 准备身份证明、学历学位证明、劳动合同、社保记录、无房证明、个人承诺书和单位审核意见。[C4]
3. 在 2026-06-01 至 2026-07-31 通过示例市人才服务平台提交材料。[C5]
4. 向区人才服务窗口确认户籍是否影响本政策申报；原文未列出户籍限制，但仍建议人工核验。
