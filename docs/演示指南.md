# 演示顺序建议

推荐以下演示顺序展示 GraphiteUI 的核心能力：

## 1. 首页

说明 GraphiteUI 定位 —— 可视化 Agent 编排工作台

## 2. Workspace

说明这是总览与入口，展示 Recent Graphs、Recent Runs 等信息

## 3. Editor

演示拖拽节点、连线、配置参数的完整流程

## 4. Validate

说明 graph 不是随便画的，需要通过合法性校验

## 5. Run

触发执行，展示后端 LangGraph runtime 如何运行

## 6. 节点状态

展示白盒可观测性 —— 节点执行状态实时高亮

## 7. 节点摘要

展示每步执行结果，了解数据流转

## 8. Runs

展示历史运行记录，说明持久化能力

## 9. Knowledge / Memories

展示资产页面，说明运行产物沉淀
