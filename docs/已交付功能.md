# 交付边界

## 已完成功能

### 核心能力
- graph 创建、保存、加载
- 节点拖拽与连线
- 节点参数配置
- preset 驱动建点与节点结构化编辑
- graph 合法性校验
- graph 运行
- 输出节点预览与基础 run 结果回填
- runs 列表与 run detail
- knowledge 和 memories 查看

### 技术实现
- 前后端可运行骨架
- 两套后端模板：`creative_factory` 与 `hello_world`
- SQLite 持久化
- Tailwind CSS 基础设施
- 主题预设系统

## 未完成功能

以下内容不在当前交付范围内：

- 完整 state-aware UI
- 参数级 socket 通用化
- editor 内完整节点执行明细面板
- 自然语言自动生成完整 graph
- 双向无损代码编译
- 子图 / Subgraph
- state diff 调试
- 人类在环在线中断编辑
- 多人协作
- 多 Agent 协同

## 环境变量

本地模型相关环境变量优先读取顺序：

| 变量名 | 别名 | 说明 |
|--------|------|------|
| `LOCAL_BASE_URL` | `OPENAI_BASE_URL` | API 基础地址 |
| `LOCAL_API_KEY` | `OPENAI_API_KEY` | API 密钥 |
| `LOCAL_TEXT_MODEL` | `TEXT_MODEL` | 模型名称 |
